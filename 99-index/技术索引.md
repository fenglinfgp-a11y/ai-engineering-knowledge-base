# 技术索引

来源类型：实战总结

## 问题场景

需要按问题快速找到对应知识文档，而不是逐个目录翻找。

## 判断方法

- Codex 指令和工作边界：看 `00-codex-prompts/`。
- Android 构建、日志、Billing、LSPosed：看 `01-android/`。
- iOS deb、rootless、dylib：看 `02-ios/`。
- 后端接口和服务日志：看 `03-backend/`。
- Vue/Nuxt 和打包：看 `04-frontend/`。
- MySQL 表结构、备份、SQL：看 `05-database/`。
- Ubuntu、Nginx、Docker、Java 服务：看 `06-server/`。
- 源码交付：看 `07-project-delivery/`。
- 公开资料来源：看 `sources.md`。
- 脱敏实战案例：看 `cases/`。
- 操作前检查清单：看 `checklists/`。
- 常用命令专题：看 `commands/`。

## 验证方法

- 能从具体问题定位到对应文档。
- 对应文档包含判断方法、操作步骤、验证方法和注意事项。

## 注意事项

- 新增文档后同步更新本索引。
- 不确定归类时，优先放到最接近的技术目录，并在索引中补充交叉入口。
