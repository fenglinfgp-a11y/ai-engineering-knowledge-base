# Linux 常用命令

来源类型：官方文档 / 实战总结

## 问题场景

服务器排查时，需要查看系统版本、磁盘、内存、进程、端口和日志。

## 适用范围

适用于 Ubuntu 等 Linux 服务器的合法运维和测试环境排查。

## 判断方法

先只读查看状态，再决定是否执行会影响服务的命令。

## 操作步骤

```bash
pwd
```

用途：查看当前目录。

```bash
ls -lah
```

用途：查看当前目录文件、权限和大小。

```bash
df -h
```

用途：查看磁盘空间使用情况。

```bash
free -h
```

用途：查看内存使用情况。

```bash
ps aux | grep <keyword>
```

用途：按关键字查找进程。

```bash
tail -n 200 <log-file>
```

用途：查看日志文件最后 200 行。风险：日志可能包含敏感信息，外发前必须脱敏。

## 相关命令

常用组合：`pwd`、`ls -lah`、`df -h`、`free -h`、`ps aux`、`tail -n`。

## 验证方法

能根据输出确认目录、资源、进程或日志状态。

## 注意事项

不要在不确认路径的情况下执行删除命令。
